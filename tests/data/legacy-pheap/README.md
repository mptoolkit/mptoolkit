Legacy pheap fixtures for serialization compatibility tests.

These archives contain sparse pheap files from the local historical `~/dmrg`
tree. They are stored as tarballs so the repository does not need to carry the
full sparse file size in the working tree.

- `old-su2-lattice.tar.gz`: `/home/ian/dmrg/old/dmrg/lat`, an SU(2) lattice
  from 2016 that exercises the historical InfiniteMPO variant layout.
- `xxz-u1-infinite-wavefunction.tar.gz`: `/home/ian/dmrg/xxz/psi`, a U(1)
  infinite wavefunction from 2021 with wavefunction format version 6.
